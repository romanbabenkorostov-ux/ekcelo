# XSD ФНС — ЕГРИП (сведения об индивидуальных предпринимателях)

## Назначение

`egrn_parser/parsers/egrul_egrip_parser.py` парсит и (через lxml) валидирует
XML-выписки ЕГРИП по XSD из этой папки.

Парсер опознаёт формат по корню `Файл`:
- `@ТипИнф` начинается на `ЕГРИП_` → реестр ЕГРИП (`ЕГРИП_ОТКР_СВЕД` /
  `ЕГРИП_ЗАКР_СВЕД` / `ЕГРИП_ОГР_СВЕД`);
- `@ВерсФорм` → версия формата (текущая — `4.07`).

`find_xsd("ЕГРИП", версия)` берёт **самый свежий** файл по сортировке имени.

## Источник

XSD публикует ФНС (ГНИВЦ) в составе требований к составу и структуре файла,
содержащего сведения из ЕГРИП. Файл — в кодировке **windows-1251**, без
targetNamespace.

- Реестр форматов ФНС: https://www.nalog.gov.ru/
- Текущая редакция в репо: `VO_RIGF_2_311_27_04_07_01.xsd` — версия формата **4.07**.

## Обновление редакции

1. Скачать новый `VO_RIGF_*.xsd`.
2. Положить **рядом** со старым (старые нужны для архивных выписок).
3. Добавить пару `("ЕГРИП", "<версия>")` в `SUPPORTED_FORMATS`
   (`egrul_egrip_parser.py`).
4. Отметить редакцию в `obsidian/Changelog/`.
