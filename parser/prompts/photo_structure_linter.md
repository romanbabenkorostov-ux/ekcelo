## Как пользоваться

1. Скопировать промпт выше (от ```` ``` ```` до ```` ``` ````).
2. Дописать в самом конце список своих фотографий — что и где сфотографировано.
3. Отправить ИИ. Он ответит:
   • деревом папок,
   • командами `mkdir`/`md` для создания пустых папок,
   • манифестом `photos_manifest.json`.
4. Создать папки этими командами и разложить файлы по местам (или
   ориентироваться на манифест).
5. Запустить `pirushin_sosn_rocha_08_build_kmz_v1.py` — KMZ соберётся с
   корректной привязкой фото к объектам.

## Что увидит парсер

| Папка | Слой в KMZ | Координата фото |
| --- | --- | --- |
| `По_объектам/<КН>/` | подгруппа "По объектам" + превью в баллоне КН | EXIF GPS → центроид КН |
| `По_оборудованию/<инв>/` | "По оборудованию" | EXIF GPS → центроид связанного КН |
| `По_BU/<slug>/` | "По BU" | только EXIF GPS |
| `00_Нераспределенные/` | "Без привязки" | только EXIF GPS (иначе пропуск) |
