# PhotoMap — EXIF геолокация на карте

Одностраничный сайт для визуализации GPS-точек из EXIF-данных JPG фотографий.

## Возможности

- Drag & drop или выбор JPG файлов
- Автоматическое чтение GPS координат из EXIF
- **Конус угла съёмки** (GPSImgDirection) на карте
- Автоподгонка масштаба под все загруженные точки
- Клик по маркеру или миниатюре — показывает фото с EXIF-данными
- Полноэкранный просмотр фото по клику
- Навигация между фото: кнопки или ← → на клавиатуре
- Переключение слоёв: OSM / Спутник (Esri)
- Работает полностью в браузере, без сервера

## Использование

Перейдите на [страницу сайта](https://YOUR-USERNAME.github.io/YOUR-REPO/)  
и перетащите JPG файлы с GPS-метками.

## Развёртывание на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите  в корень репозитория
3. Settings → Pages → Source: **main branch / root**
4. Сайт будет доступен по адресу 

## Технологии

- [Leaflet.js](https://leafletjs.com/) — карта
- [exifr](https://github.com/MikeKovarik/exifr) — чтение EXIF
- OSM / Esri — тайлы карты
- Нет сервера, нет зависимостей для установки
