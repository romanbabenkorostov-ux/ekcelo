# EkceloFoto — EXIF геолокация на карте

Одностраничный сайт для визуализации GPS-точек из EXIF фотографий.  
Работает полностью в браузере — без сервера, без установки.

🌐 **[Открыть сайт](https://romanbabenkorostov-ux.github.io/ekcelo/)**

---

## Возможности

### Загрузка фотографий
- **Drag & drop** или выбор JPG-файлов с компьютера
- **Google Drive (папка)** — множественная загрузка через API-ключ
- **URL-параметры** — автоматическая загрузка фото из ссылки (Google Drive + Яндекс.Диск)

### Карта
- Автоматическое чтение GPS-координат из EXIF
- **Конус угла съёмки** (GPSImgDirection) на карте
- Автоподгонка масштаба под все загруженные точки
- Клик по маркеру — подробная карточка фото
- Полноэкранный просмотр, навигация ← →
- Переключение слоёв: OSM / Спутник (Esri)
- **Кадастровый слой** НСПД (nspd.gov.ru) с регулировкой прозрачности
- Клик по карте → кадастровые номера участков НСПД

### Ручной GPS (новое!)
- Для фото **без GPS** — кнопка «📍 Указать точку вручную»
- Два способа задать координаты:
  - Кликнуть по карте
  - Ввести текстом: `47.373533, 39.899542` или `47°22'24.72"С 39°53'58.35"В`
- Интерактивный **слайдер угла съёмки** (0°–360°) с живым обновлением маркера
- Поддержка колёсика мыши на слайдере
- Запись координат и угла прямо в **EXIF файла** (piexifjs)
- Автоматическое скачивание модифицированного JPEG

### Поделиться (новое!)
- Кнопка **🔗 Поделиться этими фото** в шапке (появляется при наличии remote-фото)
- Формирует ссылку вида `?photo1=URL1&name1=...&photo2=URL2&name2=...`
- При открытии ссылки все фото загружаются и появляются на карте автоматически

## Форматы share-ссылок

```
# Google Drive (одиночный файл):
?photo1=https://drive.google.com/file/d/FILEID/view

# Яндекс.Диск:
?photo1=https://disk.yandex.ru/i/SHORTID
?photo1=https://yadi.sk/i/SHORTID

# Несколько фото с именами:
?photo1=URL1&name1=Мой%20снимок&photo2=URL2&name2=Другой

# Google Drive + Яндекс в одной ссылке:
?photo1=https://drive.google.com/file/d/1cEqjm.../view&photo2=https://disk.yandex.ru/i/4qmcMYt52Q_RFg&name2=Яндекс
```

> **Важно для Яндекс.Диска:** ссылка должна быть открыта «Для всех» (публичный доступ).  
> **Важно для Google Drive:** файл должен быть доступен по ссылке без авторизации.

## Поддерживаемые EXIF-поля

| Поле | Описание |
|------|----------|
| GPSLatitude / GPSLongitude | Координаты |
| GPSImgDirection | Направление съёмки (угол конуса) |
| DateTimeOriginal | Дата и время |
| GPSAltitude | Высота над уровнем моря |
