# Описание шаблона: геометрия объекта (NSPD)

**Файл:** `nspd_geometriya_TEMPLATE.json` · **Формат:** JSON · **→** `land_contours`

## Что это
Геометрия земельного участка (обычно — выход NSPD-парсера). Один файл на объект.

## Поля
- `cad_number` — кадастровый номер.
- `geojson` — геометрия: `Polygon` (1 контур = ЗУ) или `MultiPolygon` (несколько
  контуров = МКУ/ЕЗП). Координаты — `[долгота, широта]`.

## Важное
- Для **ЕЗП** тип определяется по дочерним КН из выписки, а НЕ по геометрии
  (геометрия не понизит ЕЗП до МКУ).

## Как загрузить
`land_ingest.ingest_sidecar_contours` / `01c_contours_to_db.py` → `land_contours`.
