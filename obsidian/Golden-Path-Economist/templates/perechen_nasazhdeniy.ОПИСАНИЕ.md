# Описание шаблона: перечень виноградных насаждений

**Файл:** `perechen_nasazhdeniy_TEMPLATE.txt` · **Формат:** текст (из залогового документа) · **→** `agro_parcel`/`agro_crop_cycle`

## Что это
Блоки «Многолетние насаждения (виноградные насаждения)… Предмет залога N» из
залоговых/оценочных документов. Один блок = одно насаждение.

## Поля блока (парсер ищет их по названиям)
- Предмет залога **N** → код участка (ПЗ-N)
- Индивидуальный номер в **федеральном реестре** виноградных насаждений
- **Площадь** виноградного насаждения (га)
- Сорт **привоя** (это сорт винограда) и сорт **подвоя**
- **Код сорта** в госреестре селекционных достижений
- **Год высадки**
- **Количество кустов** (можно с пробелами: «40 931»)

## Как загрузить
`vineyard_perechen.ingest_plantings(conn, текст, land_cad_by_pledge={N: "КН ЗУ"})`.
Можно копировать текст прямо из документа — поля находятся по заголовкам.
