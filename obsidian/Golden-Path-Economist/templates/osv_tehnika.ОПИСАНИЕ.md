# Описание шаблона: ОСВ — техника / основные средства

**Файл:** `osv_tehnika_TEMPLATE.xlsx` · **Формат:** Excel (выгрузка из 1С) · **→** таблица `fixed_asset`

## Что это
Оборотно-сальдовая ведомость по счетам ОС (01.x). Колонка **«Субконто»** содержит
либо **код счёта** (01.01, 01.08…), либо **наименование** основного средства под этим счётом.

## Колонки
| Колонка | Смысл |
|---|---|
| Субконто | код счёта (строка-заголовок) ИЛИ наименование ОС |
| Кол-во | количество единиц |
| Сумма (сальдо) | стоимость, руб. |

## Важное
- **Счёт 01.08** = ОКС, права на который **ещё не оформлены** (не на кадастровом учёте) →
  система помечает `on_cadastre=0` (кандидат на постановку на учёт).
- Можно вставлять реальную выгрузку 1С — парсер сам находит колонку «Субконто».

## Как загрузить
`osv_assets.parse_osv(файл.xlsx)` → реестр ОС. (В реальном пайплайне — стадия ОСВ.)
