# 2026-05-25 — SPEC экспорта в ЭТП (torgi.gov.ru / roseltorg.ru / sberbank-ast.ru)

## Итог
Подготовлена спецификация для будущей доработки экспорта объектов сюрвея в три российские ЭТП. Скрипты не писались (по запросу).

## Артефакты
- `docs/экспорт_под_торги/SPEC_etp_export.md` — основной спек (12 разделов: маппинг полей, гэп-инвентарь, миграция БД, структура модуля, verification).
- `docs/экспорт_под_торги/00..05_*.md` — исходные материалы по структуре описания и Jinja-шаблонам.

## Ключевые решения
- Scope: все 3 ЭТП сразу, общее JSON-ядро + платформенные адаптеры.
- Единица экспорта — лот (группа КН), таблицы `lots` + `lot_items`.
- Гэп-поля закрываются новой таблицей `object_etp_profile` (JSON-колонки) одной миграцией.
- Источники гэп-полей: ОСВ/survey-лист экономиста + парсер EXIF UserComment; NSPD/LLM — фолбэк.
- Артефакты на лот: `long_description.json` + `description.{short,full}.txt` + `lot_appendix.pdf`.
- Модуль: `parser/exporters/etp/` с `templates/torgi_long_description.j2` из материалов файла 05.

## Гэп-инвентарь (12 пунктов)
Компонентный адрес; окружение/транспорт; реновация/износ/благоустройство; инженерка; планировка/состояние; зонирование/спец-ограничения как блок; технические/территориальные/иные риски; advantages+notes; мебель; тип процедуры лота; группировка КН в лоты; маркировка source/confidence.
