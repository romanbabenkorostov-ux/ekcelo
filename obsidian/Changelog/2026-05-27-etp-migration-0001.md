## Решения по миграции
- **Новая директория `schema/migrations/`** — `CLAUDE.md` §3 требовала миграции через файлы, но dir не существовал. Этот PR её создаёт; нумерация `0NNN_*.sql`.
- **CHECK-constraint `lot_id` через `NOT GLOB`** — SQLite не поддерживает регулярки на уровне SQL без расширений. Эквивалент `[A-Za-z0-9_:/-]+` через `lot_id NOT GLOB '*[^A-Za-z0-9_:/-]*'` + явная проверка длины.
- **JSON-колонки как TEXT** — SQLite не имеет нативного JSON type, парсинг идёт через `json()` функции или Python-сторону.
- **`primary_cad_number` ON DELETE SET NULL** — лот сохраняется при удалении объекта (ручные пометки экономиста ценнее), но FK становится NULL.
