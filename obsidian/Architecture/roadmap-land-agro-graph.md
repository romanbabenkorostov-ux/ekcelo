- Тесты: `test_land_*` → 36 passed.

## ⏳ В плане (не сделано)

### P0 — критический путь (заблокирован образцом)
- **A. Парсер техкарты** (`agro_techcard.py`) → `agro_parcel` + `agro_crop_cycle`
  (озимая plan→fact) + `agro_event`.
  - **Блокер:** нужен 1 обезличенный образец техкарты/ОСВ-поля в `fixtures/agro/`.
  - Объём: парсер Excel/таблицы → строки + golden-тест; на ingest событий —
    прогон через `validate_event_attrs` (D уже готов).

### P1 — разблокировано, но даёт ценность после A
- **E. Вьюхи-агрегаты агро** (после A, нужны данные):
  - урожай по сортам/сезонам/полям (Σ `harvest.volume_kg`);
  - пестицидная нагрузка (разворот `active_substances[]` → Σ `rate`);
  - техсхема лота (`agro_crop_cycle` за сезон лота);
  - сроки сбора + кислотность/сахар.
- **Wire-валидация событий:** подключить `validate_event_attrs` в ingest техкарты
  (отбраковка/лог невалидных `attrs`). Зависит от A.

### P1 — земля/граф, разблокировано сейчас
- **F. Остальные рёбра графа** (GRAPH_SCHEMA, помимо контуров):
  - `located_on` — объекты на участке (`linked_objects`: land→build);
  - `right_holder` — право на объект (`rights`→`entity_registry`);
  - `asset_of` — техника на ОКС (`fixed_asset.cad_number`→`build_<cad>`).
  - Аналогично `land_graph_edges`: функции + вьюхи `v_*_graph_edges`.
- **G. Связь агро↔земля (ADR-006 §E / ADR-005):** заполнение
  `agro_parcel.land_cad`/`contour_no` при ingest; ОКС 01.08 (`on_cadastre=0`) →
  кандидат на постановку на учёт, при оформлении `cad_number` → перелинковка на
  `build_<cad>`.

### P2 — контракт с граф-слоем / просмотрщиком (кросс-команда)
- **H. Граф-рендер контуров в просмотрщике** — контракт `graph_node_id` ==
  `node.id`, UNION `v_*_graph_edges` → `v_graph_edges` в `contracts/db`
  (см. GRAPH_SCHEMA §«Что добавить»). Согласовать с соседней командой.

## Порядок (рекомендация)
1. **F** (рёбра графа located_on/right_holder/asset_of) — разблокировано, замыкает
   граф-слой земли/субъектов.
2. **A** — как только появится образец техкарты (критический путь §6).
3. **E** + wire-валидация — сразу после A.
4. **G**, **H** — по мере готовности контракта граф-слоя.
