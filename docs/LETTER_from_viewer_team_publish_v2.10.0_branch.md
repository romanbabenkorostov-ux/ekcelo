## 3. Инструкция (PowerShell / bash, ваш клон репозитория)

```bash
# 0. Положите ekcelo-v2.10.0.bundle рядом, например в C:\tmp\ (или /tmp)

# 1. Проверить целостность bundle
git bundle verify /путь/ekcelo-v2.10.0.bundle
#    Ожидаемо: "The bundle records a complete history" + строка ref
#    refs/heads/claude/fix-image-tile-gaps-e4cre <hash>

# 2. Затянуть ветку из bundle в локальный клон (под тем же именем)
git fetch /путь/ekcelo-v2.10.0.bundle \
  claude/fix-image-tile-gaps-e4cre:claude/fix-image-tile-gaps-e4cre

# 3. Убедиться, что это ровно ожидаемый коммит и он трогает только 2 файла
git rev-parse claude/fix-image-tile-gaps-e4cre
#    → c20fb56993bea81e9ac2b3447d143da612b69f7e
git show --stat --oneline c20fb56
#    → изменены только index.html и sw.js
git show --stat c20fb56 | grep -E "parser/|obsidian/" || echo "OK: ваши каталоги не затронуты"

# 4. Опубликовать ВЕТКУ на origin (НЕ force, НЕ main)
git push origin claude/fix-image-tile-gaps-e4cre
```
